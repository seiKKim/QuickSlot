## 🔧 추가 설정이 필요한 항목

### 1. 구글 서치 콘솔 등록

#### 단계:
1. **구글 서치 콘솔 접속**
   - https://search.google.com/search-console 접속
   - Google 계정으로 로그인

2. **속성 추가**
   - 속성 추가 → URL 접두어 입력: `https://quickslot.co.kr`
   - 소유권 확인 선택

3. **HTML 태그 방식 선택**
   - 소유권 확인 방법 선택
   - 제공된 메타 태그 코드 복사
   - 예: `<meta name="google-site-verification" content="xxxxxxxxxxxxx" />`

4. **layout.tsx 수정**
   ```tsx
   verification: {
     google: '복사한-인증코드', // 여기에 실제 코드 입력
   },
   ```

5. **확인 및 제출**
   - 확인 버튼 클릭
   - sitemap 제출: `https://quickslot.co.kr/sitemap.xml`

---

### 2. 네이버 서치어드바이저 등록

#### 단계:
1. **네이버 서치어드바이저 접속**
   - https://searchadvisor.naver.com/ 접속
   - 네이버 계정으로 로그인

2. **사이트 등록**
   - 웹마스터 도구 → 사이트 등록
   - 사이트 URL 입력: `https://quickslot.co.kr`

3. **소유권 확인**
   - HTML 태그 방식 선택
   - 제공된 메타 태그 코드 복사
   - 예: `<meta name="naver-site-verification" content="xxxxxxxxxxxxx" />`

4. **layout.tsx 수정**
   ```tsx
   verification: {
     other: {
       'naver-site-verification': '복사한-인증코드', // 여기에 실제 코드 입력
     },
   },
   ```

5. **확인 및 제출**
   - 확인 버튼 클릭
   - 사이트맵 제출: `https://quickslot.co.kr/sitemap.xml`

---

### 3. 메타 태그 최적화

#### 현재 설정 확인:
- `src/app/layout.tsx` 파일에서 다음 항목들이 설정되어 있는지 확인:
  - `title`: 페이지 제목 (검색 결과에 표시)
  - `description`: 페이지 설명 (검색 결과에 표시)
  - `keywords`: 검색 키워드
  - `canonical`: 정규화 URL

#### 개선 권장사항:
- 각 페이지별로 고유한 메타 태그 설정 (about, contact, reviews 등)
- 검색 키워드 최적화
- 설명 텍스트는 150-160자 이내로 유지

---

### 4. 구조화된 데이터 추가

#### 현재 상태:
- Organization 스키마만 추가됨

#### 추가 권장 스키마:
- **LocalBusiness**: 지역 비즈니스 정보
- **Service**: 제공 서비스 정보
- **FAQPage**: FAQ 페이지
- **Review**: 고객 리뷰
- **BreadcrumbList**: 네비게이션 경로

#### 예시 코드:
각 페이지에 필요한 스키마를 추가할 수 있습니다. 예를 들어:
- `/services/camping` 페이지에는 `Service` 스키마 추가
- `/reviews` 페이지에는 `Review` 스키마 추가

---

### 5. 콘텐츠 최적화

#### 권장사항:
1. **고품질 콘텐츠 작성**
   - 검색 사용자의 질문에 답하는 콘텐츠
   - 자연스러운 키워드 사용
   - 500자 이상의 의미 있는 콘텐츠

2. **헤딩 태그 사용**
   - H1: 페이지당 1개만 사용
   - H2, H3: 섹션 구분을 위해 사용
   - 검색 키워드를 헤딩에 포함

3. **이미지 최적화**
   - Alt 텍스트 추가
   - 파일명에 키워드 포함
   - 적절한 이미지 크기 (로딩 속도 고려)

4. **내부 링크 구조**
   - 관련 페이지로 자연스러운 링크 연결
   - 중요한 페이지는 홈페이지에서 바로 접근 가능하도록

5. **모바일 최적화**
   - 반응형 디자인 (이미 구현됨)
   - 모바일 친화적 사이트 확인 (구글 서치 콘솔에서 테스트)
